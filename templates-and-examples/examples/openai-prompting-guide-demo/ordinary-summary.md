# Ordinary Summary Baseline

The OpenAI Prompting guide explains that prompts influence model output and should be improved through testing and iteration. Prompt engineering can involve clear instructions, examples, evaluation, and versioning. Good prompts help models produce more useful and reliable responses.

## Why This Baseline Is Weaker

- It compresses the source into a single paragraph.
- It does not create reusable notes.
- It does not preserve a card-level source trail.
- It does not produce links between ideas.
- It does not show how the material can become an article or knowledge asset.

The Zettelkasten dry-run turns the same source snapshot into cards, links, a draft, and a quality report.

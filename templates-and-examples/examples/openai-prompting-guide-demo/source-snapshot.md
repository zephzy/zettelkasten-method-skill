# Source Snapshot: OpenAI Prompting Guide

This demo uses the official OpenAI Prompting guide as a public AI-learning source.

- Source title: Prompting guide
- Source origin: OpenAI developer documentation
- Source URL: https://developers.openai.com/api/docs/guides/prompting
- Source type: official technical guide
- Demo purpose: show how the Zettelkasten Method Skill turns a prompt-engineering guide into reusable learning notes and a short article.

## Source Summary Used For This Demo

The guide explains that prompt quality can strongly shape model output, that improving prompts is an iterative process, and that production prompt work benefits from evaluation and version control. It also presents prompting as a practical engineering activity: write instructions, test outputs, measure whether the result works, and revise.

This repository does not reproduce the guide. It uses a short paraphrased learning snapshot with a source URL so the generated notes remain traceable.

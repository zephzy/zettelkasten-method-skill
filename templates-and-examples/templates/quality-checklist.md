# Quality Checklist

Use this checklist after generating notes and drafts.

## Card Quality

- [ ] Each permanent note has one main idea.
- [ ] Permanent notes are written in the user's own words.
- [ ] Literature notes preserve source information.
- [ ] Fleeting notes are allowed to be rough but still meaningful.
- [ ] No card depends on a long copied passage.

## Link Quality

- [ ] Each proposed link explains the relationship.
- [ ] Links are more specific than tags.
- [ ] At least one link shows cause/effect, tension, application, or analogy.

## Draft Quality

- [ ] The draft is built from note IDs.
- [ ] Important claims can be traced back to notes.
- [ ] The draft does not introduce unsupported claims.
- [ ] The writing is clear rather than merely fluent.

## Revision Triggers

Revise if:

- A permanent note contains two ideas.
- A link only says "related to".
- The draft could have been written without the notes.
- Source metadata is invented.

中文说明：质量检查的目标不是让文本更花哨，而是保证卡片系统可积累、可追溯、可复用。

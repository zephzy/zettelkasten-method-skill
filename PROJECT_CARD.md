# Project Card: Zettelkasten Method Skill

## One-line Goal

Create a bilingual, GitHub-ready prompt pack that turns raw learning material into Zettelkasten-style cards, links, outlines, drafts, and quality reports.

## Positioning

This is a methodology-distillation skill, not a generic summarizer. It packages an expert workflow into a repeatable AI-agent process.

## Target Users

- AI learning note takers.
- Obsidian / Markdown users.
- Students and researchers.
- Writers who want grounded drafts from accumulated notes.

## v0.1 Scope

- Platform-neutral prompt pack.
- Markdown templates.
- AI-learning sample source.
- Complete dry-run output.
- Before/after demo.
- Quality checklist.

## Out of Scope

- Python validation scripts.
- Direct Codex skill installation.
- Direct Claude Code packaging.
- Web app UI.

## Success Criteria

- A GitHub visitor understands the value in 30 seconds.
- A user can copy one prompt and run the workflow immediately.
- The example output proves the project is more than a summary prompt.
- The workflow enforces atomicity, traceability, linkability, own-words rewriting, and bottom-up synthesis.

## Interview Talking Points

- Methodology distillation: turning tacit expert workflow into executable agent steps.
- Prompt system design: explicit stages, output contracts, and quality gates.
- Knowledge workflow design: source-to-card-to-article pipeline.
- GitHub product packaging: README, templates, demo, roadmap.
